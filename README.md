# \<og-image-element>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i og-image-element
```

## Usage
```html
<script type="module">
  import 'og-image-element/og-image-element.js';
</script>

<og-image-element></og-image-element>
```
